```
leo new ojaaay_auction
```

<p>3. Step 3: Compile and run the program</p>

```
# build & setup & place bid - leo run place_bid aleo1mwndcw7y7p3letp6lcjxlk2nyzddxfhxfxpla7czz65g0u7cpqyqusx99h 100u64
```
